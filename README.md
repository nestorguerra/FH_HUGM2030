# InformeFH_HUGM

**Plan Estratégico de IA 2026–2028 · Servicio de Farmacia Hospitalaria · HGUGM**
Experiencia de lectura interactiva del informe, en un único archivo `index.html` autocontenido: fuentes, estilos, datos y lógica van incluidos. Nota: carga React desde CDN (unpkg.com), por lo que requiere conexión a internet — sin problema para GitHub Pages.

> Confidencial · uso restringido al Servicio. Si el contenido es sensible, use un repositorio **privado** (GitHub Pages en repos privados requiere plan de pago) o valore otra vía de distribución.

## Cómo publicarlo en GitHub Pages

1. Cree el repositorio `InformeFH_HUGM` en su cuenta de GitHub (botón **New repository**).
2. Suba los archivos de esta carpeta (`index.html` y este `README.md`): **Add file → Upload files → Commit changes**.
3. Active Pages: **Settings → Pages → Source: Deploy from a branch → Branch: `main` / carpeta `/ (root)` → Save**.
4. En 1–2 minutos la web estará en:
   `https://SU-USUARIO.github.io/InformeFH_HUGM/`
